# Instructions

1. Navigate to `/backend`

2. Create your `venv`

```console
python -m venv venv
```

3. Activate `venv`

   `Macos/linux`

   ```console
   source venv/bin/activate
   ```

   `Windows`

   ```console
   venv\Scripts\activate
   ```

4. Install dependencies

```console
pip install -r requirements.txt
```


5. Run the API

```console
python app.py
```
6. Run the frontend
